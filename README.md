A simple textbot for persian langugae


## Developement


Install and prepare a virualenv based on this link:
https://virtualenvwrapper.readthedocs.io/en/latest/

```bash
git clone <git-url>
cd sunrise
workon sunrise
pip install -e .
pip install -r requirements-dev.txt
pytest 
```
